Industry Git Workflow
git clone→ git checkout -b feature/xyz→ git add . && git commit→ git push -u origin feature/xyz→ Open Pull Request→ Code Review→ git merge --no-ff→ git tag v1.x.x
main — protected feature branches never force push to main git reset --hard = destructive prefer revert in shared repos use --force-with-lease not --force
Commands reference
Setup & Config 5 cmds
git config --global user.name "Name"Set global username
git config --global user.email "e@mail"Set global email
git initInit new repo
git clone <url>Clone remote repo
git config --listShow all config
Branching 6 cmds
git branchList branches
git branch <name>Create branch
git checkout <name>Switch branch
git checkout -b <name>Create & switch
git branch -d <name>Delete branch
git branch -m <old> <new>Rename branch
Staging 6 cmds
git statusCheck working tree
git add <file>Stage specific file
git add .Stage all changes
git add -pStage interactively
git restore --staged <file>Unstage file
git diff --stagedDiff staged changes
Committing 5 cmds
git commit -m "message"Commit with message
git commit -am "message"Stage tracked + commit
git commit --amendEdit last commit
git log --onelineCompact history
git log --graph --allVisual branch graph
Remotes 6 cmds
git remote -vList remotes
git remote add origin <url>Add remote
git fetch originFetch all branches
git pull origin <branch>Pull & merge
git push origin <branch>Push branch
git push -u origin <branch>Push & set upstream
Merging & Rebasing 5 cmds
git merge <branch>Merge into current
git merge --no-ff <branch>Merge with commit
git rebase <branch>Rebase onto branch
git rebase -i HEAD~NInteractive rebase N commits
git cherry-pick <hash>Apply specific commit
Undo & Reset 5 cmds
git revert <hash>Safe undo (new commit)
git reset --soft HEAD~1Undo commit, keep staged
git reset --mixed HEAD~1Undo commit, keep files
git reset --hard HEAD~1Discard commit & changes
git restore <file>Discard working changes

git stash apply stash@{n}Apply specific stash

Inspect & Compare 5 cmds

git show <hash>Show commit details
git blame <file>Who changed what
git bisect startBinary search bug
